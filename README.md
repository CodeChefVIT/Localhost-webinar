## Dependencies
 - FastAPI
 - Docker
## Instructions

#### Directions to Install
```bash
$ git clone https://github.com/CodeChefVIT/Localhost-webinar
$ cd Localhost-webinar
$ pip3 install -r requirements.txt
```
#### Directions to Run
```bash
$ docker compose up
```
